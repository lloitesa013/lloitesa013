# Reproduction Boundary

This kit is a sanitized reproducibility surface. It tells a reviewer what would need to be reproduced without publishing the mixed raw workspace.

## Included In This Kit

- Scenario definitions.
- Frozen anchor identifiers.
- Primary result table.
- Limitations and non-claims.
- Public source boundary.

## Not Included

- CARLA installation.
- Virtual environments.
- Generated logs and caches.
- Raw datasets, checkpoints, and model artifacts.
- Raw videos.
- Handoff notes.
- External baseline repos.
- Private local paths.

## Future Public Package Requirements

A standalone public package should include:

- exact CARLA version and map
- scenario scripts or sanitized scenario descriptions
- seed list
- metric definitions
- validation report for seed/scenario matching
- result tables
- failure-analysis section
- non-claim section
