# Research Kits

This folder contains sanitized public research kits for the autonomous-driving line behind the portfolio.

These are not raw workspaces, paper rewrites, or deployment packages. They are compact reviewer-facing packets that explain the architecture, evidence, limitations, and reproduction boundary without exposing private local material.

Read the source boundary first: [CARLA Source Boundary](CARLA_SOURCE_BOUNDARY.md).

## Kits

- [X-MoD Visual Paper Kit](xmod-visual-paper-kit/README.md): explainable four-drive routing and gate-event alignment evidence.
- [Angelos Reproducibility Kit](angelos-reproducibility-kit/README.md): protocol-frozen CARLA evidence for a top-level judgment layer.
- [Legacy Experiment Map](LEGACY_EXPERIMENT_MAP.md): how the private numbered CARLA runs relate to the public sanitized packets.

## Boundary

- Raw CARLA workspaces, logs, videos, generated files, and local scripts are not included.
- Admissions material and personal context are not included.
- Each kit states what it does and does not claim before pointing to results.
- Public validators run only on sanitized packet files committed here.
